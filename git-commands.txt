==============GIT CONFIG============
1. Set global username   : git config --global user.name "username"
2. Check global username : git config --global user.name

3. Set global email      : git config --global user.email "email.com"
4. Check global email    : git config --global user.email



======== GIT BASIC COMMANDS ========


0. Initial local repository : git init
1. Clear git caches:  git rm -r --cached

---GIT BRANCH---
2. Create branch: branch [branch name] 
3. See all branches : git branches
4. Switch to another branch : git checkout [branch name]
5. Rename current branch: git branch -m [new branch name]
6. Delete branch: git branch -d [branch name]

---GIT COMMITS---
7. Take files to staging area : git add [file name] || git add .
8. Commit files : git commit -m [commit messages in string]
9. See commits history  : git log || git log --oneline

---REMOTE REPOSITORY---
10. Add remote repository: git remote add origin [repository link]
11. Push to remote repository : git push -u [remote repository name] [local repository name]
12. Remove remote repository: git remote rm [repository repository name]
